# Atari-Gym-Q_Learning-Practice


This is my attempt at a reinforcement algorithm utilizing the "MsPacman-ram-v0" environment. Throughout the project Q learning is used to determine whether to explore or exploit the action_space in each episode. 


## Install ALE Gym Package
```pip install gym[atari]```

## The Atari Environment is attributed to the Atari Learning Environment (ALE)
M. G. Bellemare, Y. Naddaf, J. Veness and M. Bowling. The Arcade Learning Environment: An Evaluation Platform for General Agents, Journal of Artificial Intelligence Research, Volume 47, pages 253-279, 2013.
